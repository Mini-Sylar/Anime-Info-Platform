About the website

Hi,Terence Quashie here. This site was created as a way to easily search for all anime as well as receive recommendations. The primary goal of this project was to learn how to hit APIs and return some data, fortunately or unfortunately the source that was used had their API written using GraphQl AniList great site btw, so this was another great learning opportunity since most people use GraphQl anyway. Features

  • Search - Search for any anime, if it exists on anilist, you'll find it
  • Descriptions - Get full synoposis on any show that you search as well as the number of episodes, rating as well as the year the show aired
  • Recommendations - Based on what you search for, you'll find 10 more shows just like it!
  • Surprise Me - Undecided on the show to start watching? just hit the surpise button and you'll find 10 random shows you can check out
  • More Info - Redirect to anilist where you can find more information such as voice actors, directors etc
  • Share - Share shows to everyone using the share button next to the anime rating

For more information and progress on this project visit Anime Info Platform on Github
